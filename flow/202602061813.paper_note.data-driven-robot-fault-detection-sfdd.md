# 単一論文ノート

## TL;DR
- 生成モデルを用いたサンプルベース故障検知/診断（SFDD）をロボットに適用し、故障分類と診断性能を検証。
- 実機（物流ロボット）とシミュレーションで、精度/再現率の改善を報告。

## 背景/貢献
- モデルベース診断の難しさを踏まえ、データ駆動な異常検知・診断を強化。
- 生成モデルで観測分布を表現し、診断を一体化した点が特徴。

## 方法/設定
- 生成モデル（RBM）で観測分布を表現し、再構成・サンプル化から異常スコアを推定。
- スライディングウィンドウの相関係数や残差の閾値化を用いた故障検知と比較。

## 結果/制約
- 実機（物流ロボットの車輪切断故障）で精度88.6%/再現率75.6%を報告。
- 既存手法より高いFDD性能（検知/診断）を示すと報告。
- 実環境での長期運用や分布シフト耐性は今後課題。

## 気づき/疑問
- 故障診断の「説明可能性」をどう担保するかが現場運用に影響しそう。
- 生成モデルの更新頻度と運用コストのバランス設計が必要。
